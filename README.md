# e-court
A web scrapper which is made to work in commandline from TN e-courts
## Requirements
Python> 3.11
## How to use
1. Run the Check_CNR
    - This will return your details + whether it is today, tomorrow, or neither.

2. Run the dts-selector.py  
    - Your selections will be saved as json 

3. Run the handler_x10.py
    - Your cause list will be written as text file
